Low-level control libraries of the actuators.  High-level control
libraries go in the controllers directory.


# ESAT_ADCSLED

For low-level control of the heartbeat LED.


# ESAT_Magnetorquer

For low-level control of the magnetorquers.


# ESAT_Wheel

For low-level control of the reaction wheel.
